## Live View Documents

Use these commands to live view documents:

```sh
$ yarn install
$ yarn start
```

Then you should go to [http://localhost:8080](http://localhost:8080) or the site displayed in output.

To build static website, run:

```sh
$ yarn build
```

Site will be generated in `_site/`
